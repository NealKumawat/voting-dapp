# 🗳️ Soroban Voting DApp

## 📌 Project Description
This project is a decentralized voting application (DApp) built on the Stellar Soroban smart contract platform. It allows users to vote for predefined candidates in a transparent and tamper-proof manner using blockchain technology.

## ⚙️ What It Does
- Initializes a voting system with a list of candidates
- Allows users to cast votes for their preferred candidate
- Stores votes securely on-chain
- Provides real-time vote counts
- Ensures fairness and transparency by leveraging decentralized infrastructure

## ✨ Features
- 🧾 On-chain vote storage
- 🔒 Tamper-proof and transparent voting
- ⚡ Fast and low-cost transactions on Stellar
- 👥 Multiple candidate support
- 📊 Retrieve individual and total vote counts
- 🧩 Simple and extensible smart contract design

## 📦 Functions Overview
- `init(candidates)` → Initialize candidates
- `vote(candidate)` → Cast a vote
- `get_votes(candidate)` → Get votes for a candidate
- `get_all()` → Get all voting results
